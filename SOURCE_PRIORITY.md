# Source Priority Policy

## TIER 1
- Official Documentation
- ArXiv
- Hugging Face
- Papers With Code
- Official Model Providers

## TIER 2
- GitHub Repositories
- Company Blogs
- Research Blogs

## TIER 3
- Community Resources
- Aggregators
- Third-party Lists

## RULES
Always prefer higher tier sources. Never overwrite Tier 1 data using Tier 2 or Tier 3 data.
